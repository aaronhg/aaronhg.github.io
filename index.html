具備十年以上前端與 Web 遊戲開發經驗。交付四十餘款遊戲，但更多時間花在讓其他人也能快速交付，設計共用架構、建立開發流程、帶新人從零到獨立。

一、寶成

維護公司內部工具集的 JavaScript 層，以 DB Schema 驅動前端畫面生成與驗證邏輯：

  1. 以 TypeScript 定義檔規範對外 API，提供明確的型別提示
  2. 針對大量資料的表格渲染卡頓，重新設計渲染策略以改善操作流暢度
  3. 重構多層連動下拉選單等複雜元件的事件觸發機制
  4. 負責 QA、文件撰寫、開發者教育訓練與除錯支援

二、騰宇

以 Rollup + Pixi.js + GSAP 開發十款單人小遊戲，涵蓋棋牌、猜拳、轉輪、跑燈及老虎機類型，於澳門 G2E 展覽中獲得熱門度排名第一。

三、京赫

以 Cocos Creator + TypeScript 開發 Web 遊戲，交付數款棋牌、三十餘款老虎機、六款捕魚機。專案量上來後，逐步從開發擴展到架構設計、跨部門流程與新人訓練。

共用架構與工具
  1. 設計並維護老虎機與魚機兩套公版架構，老虎涵蓋滾輪機制、動畫流程與 UI 狀態管理，魚機涵蓋碰撞與資料流
  2. 完成 Cocos 2 升 Cocos 3 大版本升級及新版老虎機公版架構設計
  3. 開發美術字編輯器、Spine 預覽工具、多語系預覽工具，減少企劃、美術與前端的來回確認成本
  4. 將遊戲內說明頁從 Canvas 改寫為網頁版，解決部分語系文字換行錯誤，以資料驅動組件生成畫面，提升維護彈性
專案管理與跨部門協作
  1. 主導重建遊戲專案 Jira 工作項樣板，定義狀態流程、工作項關聯、主管檢查單與會議記錄規範，與各部門 PM 確認後逐步上線
  2. 擔任美術需求清單 PM，統一素材格式、尺寸與檔名規則，作為前端與企劃、美術之間的對接窗口
新人訓練與團隊教學

先後帶過三位新人，其中一位從零基礎到能獨立承接完整專案：

  1. 基礎課程：TypeScript、Git、Promise → 節點系統、動畫、碰撞、效能優化 → 完成簡單老虎機實作，每單元搭配練習題與現場示範錄影
  2. 進階課程：公版說明、Slot 生命週期、流程控制、UI 資料綁定系統
  3. 新人第一款專案由資深成員帶著走完整開發流程，確保實際上手後能銜接獨立作業

四、個人
